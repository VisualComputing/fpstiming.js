    TimingHandler

    A timing handler holds a TimingHandler#timerPool and an TimingHandler#animatorPool. The timer pool are all the tasks scheduled to be performed in the future (one single time or periodically). The animation pool are all the objects that implement an animation callback function. For an introduction to FPSTiming please refer to fpstiming

    public frameRate(): number source

    Returns the approximate frame rate of the software as it executes. The initial value is 10 fps and is updated with each frame. The value is averaged (integrated) over several frames. As such, this value won't be valid until after 5-10 frames.
